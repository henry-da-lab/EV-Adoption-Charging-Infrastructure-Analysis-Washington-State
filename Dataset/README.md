### Data Sources

The datasets used in this project are publicly available and can be accessed from the following sources:

- **Electric Vehicle Specifications 2025 Dataset (Kaggle)**  
  https://www.kaggle.com/datasets/urvishahir/electric-vehicle-specifications-dataset-2025

- **Alternative Fuel Stations API (NREL / U.S. Department of Energy)**  
  https://developer.nrel.gov/docs/transportation/alt-fuel-stations-v1/

- **Washington State Electric Vehicle Population Data (Department of Licensing)**  
  https://catalog.data.gov/dataset/electric-vehicle-population-data

Due to dataset size and licensing considerations, raw data files are not included in this repository.
